== README ==

This app was developed as a way to post meme of the days for a class.

Tech Stack:

* Ruby on Rails

* HTML/CSS

* Bootstrap

* SQLite3

Dependencies:

* Devise

* Mini-Magick

* Carrier Wave

== Deployment Instructions ==

1. Clone the repo
2. Open the directory containing the files
3. Run rails s in your terminal
4. Open browser to the local host location
